# reddybalajimadha.com

Personal portfolio of Reddy Balaji Madha — backend software developer working on
production APIs, offline edge AI, and healthcare data systems.

- **Live site:** https://reddybalajimadha.com (deployed on Vercel)
- Static site: single `index.html` with inline styles, jQuery + Owl Carousel + Typed.js from CDNs
- Images live in `assets/`

## Deploy

Connected to Vercel; deploy manually with:

```
vercel deploy --prod
```
